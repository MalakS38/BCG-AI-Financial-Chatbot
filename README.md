# BCG-AI-Financial-Chatbot (BCG GenAI Simulation)
AI-powered financial chatbot analyzing 10-K and 10-Q reports 

This project is an AI-powered financial chatbot built as part of a **BCG GenAI job simulation**.

It extracts, analyzes, and interprets financial data from real company 10-K and 10-Q reports and provides user-friendly financial insights via a chatbot interface.

## Project Overview

I was tasked with:

- Extracting financial data from SEC 10-K and 10-Q filings
- Performing financial trend analysis
- Calculating growth metrics
- Developing a rule-based AI chatbot that answers financial queries

Companies analyzed:
- Microsoft
- Apple
- Tesla

## Tools Used

- Python
- Pandas
- Matplotlib
- SEC EDGAR financial filings

## Key Features

- Revenue Growth Calculation
- Net Income Trend Analysis
- Asset Growth Tracking
- Debt Ratio Computation
- Operating Cash Flow Analysis
- Interactive Chatbot with predefined financial queries

Example questions:
- “What was Microsoft’s operating cash flow in 2024?”
- “Did Tesla’s profit increase last year?”
- “Show Apple’s revenue growth.”

## Key Insights

- Microsoft showed the strongest financial performance (high revenue and cash flow growth)
- Tesla experienced a sharp net income decline in 2024
- Apple showed high debt ratios and weaker growth
- Microsoft had the healthiest financial ratios overall

## How to Run

1. Install requirements: pip install pandas matplotlib
2. Run the chatbot: python src/chatbot.py
